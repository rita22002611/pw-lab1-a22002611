    HTML5 e CSS

    O que é CSS

    É uma linguagem usada para estruturar r apresentar contiudos. E a quinta e mais recente versao do HTML(suportada pela apple, google, mozila e microsoft)

    HTML
    Elemento Utilizado Função
    Head ✔️ Define o cabeçalho do documento
    Body ✔️ Define o corpo do documento
    Footer ✔️ Define o rodapé do documento
    Hx ✔️ Define cabeçalhos e Tilulos
    Br ✔️ Define uma linha em branco
    P ✔️ Define um paragrafo
    Hr ✔️ Define uma linha continua ao longo da pagina
    Figure ✔️ Permite agrupar um elemento unico com uma legenda
    Table ✔️ Define uma tabela
    Thead ✔️ Permite agrupar header de uma coluna
    Tbody ✔️ Contem a parte principal da tabela
    Iframe ✔️ permite incorporar outro documento no documento
























    O que é CSS

    E uma linguagem de estilo utilizada para descrever a apresentacao de um documento escrito em uma linguagem como o html

    Tipos de Seletores
    Seletor Utilizado Função
    Universal ✔️ Aplica o estilo a todos os elementos da pagina
    Tipo ✔️ usa o nome do marcador a estilizar
    Classe ✔️ Associa um nome a um estilo
    Id ✔️ Permite identificar univocamente um elemento
    Atributo ✔️ Aplicam-se atributos a elementos
    Pseudo-classe ✔️ Associado ao estado de um elemento
    Pseudo-Elementos ✔️ Aplicam-se a partes de elementos

















    Combinsdores de Seletores
    Combinadores Utilizado Função
    Descendentes ✔️ são sepadados por espaço
    Filhos ✔️ Através do uso de ">"
    adjacentes ✔️ Através do uso de "~"
    imediatamente adjacentes ✔️ Através do uso de "+"
